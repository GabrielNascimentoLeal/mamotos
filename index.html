Configure o Firebase

O site precisa do Firebase para funcionar. Siga os passos abaixo:

  1. Acesse console.firebase.google.com
  2. Crie um projeto e ative Firestore, Storage e Auth
  3. Vá em Configurações → Aplicativos → Web → copie as credenciais
  4. Cole as credenciais no arquivo firebase.config.js
  5. Recarregue a página
